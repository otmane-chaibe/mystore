## üõ† Setup

Both a moltin and Stripe account are needed for this store to run successfully.

Create a `.env` file at the project root with your moltin `client_id` and Stripe test `publishable key`.

```dosini
MOLTIN_CLIENT_ID=
STRIPE_PUBLISHABLE_KEY=
```

`npm install`

`npm run dev`
